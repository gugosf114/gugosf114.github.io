# My Baking Creations Website

Complete multi-page website for My Baking Creations bakery.

## Pages Created:
1. **index.html** - Home page with hero, services preview, and trusted companies
2. **about.html** - Yana's story, process, and bakery history
3. **gallery.html** - Main gallery page with category navigation
4. **gallery-cakes.html** - Custom cakes gallery
5. **gallery-cookies.html** - Custom cookies gallery
6. **gallery-cakepops.html** - Cake pops gallery
7. **gallery-cupcakes.html** - Cupcakes gallery
8. **corporate.html** - Corporate orders page
9. **contact.html** - Contact info and Google Form order request

## Files:

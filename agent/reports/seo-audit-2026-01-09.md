### 2. **Missing Width/Height Attributes on Images**
**Severity**: HIGH
**Impact**: Core Web Vitals (CLS - Cumulative Layout Shift)
**Found**: 748 total `<img>` tags across 31 files, only 102 have width/height attributes

**Issue Details**:
- **646+ images missing explicit dimensions** (~86% of all images)
- Causes page content to jump during loading
- Negatively impacts Google's Core Web Vitals score
- Hurts user experience and SEO rankings

**Critical Pages**:
- `index.html` - 36 width/height found vs 40 total images
- `gallery-cupcakes.html` - 2 vs 82 images
- `gallery-cookies-hand-piped.html` - 2 vs 79 images
- `gallery-corporate-cookies.html` - 2 vs 56 images
- `corporate.html` - 5 vs 34 images

**Fix Required**:
```html
<!-- BAD -->
<img src="cake.jpg" alt="Wedding cake">

<!-- GOOD -->
<img src="cake.jpg" alt="Wedding cake" width="400" height="300">
```
